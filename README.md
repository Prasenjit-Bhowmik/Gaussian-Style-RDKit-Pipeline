# Gaussian-Style-RDKit-Pipeline

A reproducible, browser-based cheminformatics pipeline for generating 2D/3D molecular structures, calculating descriptors, energies, and physicochemical properties from SMILES. Interactive interface via Gradio on Binder; batch SMILES upload supported.

## Features
- Generate 2D and 3D structures
- Calculate 2D & 3D descriptors (Mordred)
- Calculate energies and physicochemical properties
- Batch SMILES upload or copy-paste input
- Download results as CSV + MOL files in a ZIP
- Gaussian-style workflow (QM optional/future)

## Usage
1. Open Binder (click badge below)
2. Upload `.smi` file or paste SMILES
3. Click **Run Pipeline**
4. Download results ZIP
